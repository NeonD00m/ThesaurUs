# ThesaurUs

This project will be an AI-centered grammatical and word-choice suggestion engine, including a front-end page as well as a hosted back-end.

The first version of the back-end can be found at the branch **'backend'**
- I would like to eventually do a revamp for speed, efficiency, and to use lower-level languages in another branch too.
The front-end will go in this **'main'** branch
- I plan to eventually create another branch for a heavily modified chrome extension edition
